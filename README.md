## 🚀 Featured Projects

### 🫁 [Automated Chest X-ray Analysis](https://github.com/1Taksh/Automated-Chest-Xray-Analysis)
A **multi-label CNN** model trained on the **NIH ChestX-ray dataset** to predict thoracic diseases like pneumonia, tuberculosis, and more. It outputs probabilities for each disease, handling imbalanced datasets and multi-label classification.

**Key Features:**
- Multi-label classification (14 disease categories)
- Handles overlapping conditions
- Real-world imbalanced dataset
- Sigmoid output for each disease

**Tech Stack:**
- Python, TensorFlow / PyTorch
- OpenCV, NumPy, Pandas
